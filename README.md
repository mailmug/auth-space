## Description

**Build beautiful login and registration experiences for WordPress without the complexity.**

AuthSpace is a modern authentication toolkit for WordPress that provides AJAX-powered login and registration forms, real-time field validation, and a developer-friendly API for extending user authentication.

Whether you're building a membership site, SaaS application, community, or custom onboarding flow, AuthSpace gives you a solid foundation for user authentication.

### Why Choose AuthSpace?

- Modern AJAX registration and login forms
- Real-time username and email validation
- Fast and lightweight architecture
- Extensible hooks and filters for developers
- REST API powered
- Built with modern JavaScript and Alpine.js
- Clean, developer-friendly codebase
- Translation ready

### Perfect For

* Membership websites

* Custom user onboarding
* Headless WordPress projects

## Features

### Real-Time Validation

Instantly validate usernames and email addresses without reloading the page.

### Developer Friendly

Extend and customize behavior using WordPress actions and filters.

### Lightweight

No unnecessary frameworks or page builders. AuthSpace focuses on speed and simplicity.

### Extensible Architecture

Customize registration rules, validation logic, and user workflows with ease.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/auth-space` directory, or install the plugin through the WordPress plugin installer.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Start building modern authentication experiences with AuthSpace.
